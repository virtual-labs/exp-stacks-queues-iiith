1 Demonstration artefact for Stacks

1.1 Features of the artefact

  • Artefact provides demo for stack basic operations.
  • User can see a stack(i.e stack A) of random generated numbers and stack B and stack C are provided empty.
  • At the end of demonstration , prime numbers from stack A are popped from stack A and pushed into stack B and similarly composite numbers from stack A are popped from stack A and pushed into stack B.
  • User can click the Reset button to generate a random stack A.
  • User shall click the Start button to start the demo.
  • User can slide a Slider to adjust speed of the demo as per his/her convinience.
  • User can pause the demo by clicking on the Pause button.
  • User can pause and then click on Next manually to view the demo at his/her own speed.

2.2 Body Section Elements

2.2.1 Instruction Box

This is an instruction box, in which the instructions needed for the user to perform the experiments are displayed in a form of list. 

<div class="instruction-box">
  <button class="collapsible">Instructions</button>
  <div class="content"> 
    <ul>
      <li>Click on the <b>Start</b> button to start the
      demo.</li>
      <li>Move the <b>slider</b> to adjust the speed of the
      demo.</li>
      <li>Click on the <b>Pause</b> button if you want to
      stop and manually click the Next button to have
      a step by step realization of the process.</li>
      <li>Click on the <b>Reset</b> button to start all over
      with a new set of random numbers!</li>
    </ul>
  </div>
</div>

2.2.2 Question Holder

This will hold the question of Queue Demo artefact. 

<div id="question">
  <b>Observe:</b> Stack A with random numbers is
  given. Observe how the prime numbers from stack A are
  popped and pushed into stack B whereas the others are
  pushed into stack C.
</div>
