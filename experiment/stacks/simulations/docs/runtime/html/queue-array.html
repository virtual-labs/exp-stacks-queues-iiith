1 Practice artefact for Queues using arrays

1.1 Features of the artefact

  • Artefact provides practice for queues using arrays.
  • User can see an empty array in the beginning.
  • User can click the Enqueue button to push the entered element at the rear end of queue.
  • User can click the Dequeue button to pop the element from front end of the queue.
  • User can click the clear button to clear the queue.
  • Artefact provides step by step feedback to user
  • If user's step is wrong, error message with appropriate hint is displayed. Else, step is perfromed.

2.2.1 Instruction box

This is an instruction box, in which the instructions needed for the user to perform the experiments are displayed in a form of list. 

<div class="instruction-box">
  <button class="collapsible">Instructions</button>
  <div class="content">
    <ul>
      <li>To add the elements at the end of
        Array,use <b>Enqueue</b> button.
      </li>
      <li>To delete the
        elements from the beginning of Array
        use <b>Dequeue</b> button.
      </li>
      <li>Enqueue happens at the <b>Rear</b> pointer, and
        dequeue happens from the <b>Front</b> pointer.
      </li>
      <li>Click <b>Reset</b> to clear the Queues (Array).</li>
    </ul>
  </div>
</div>
