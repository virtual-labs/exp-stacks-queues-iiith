1 Exercise artefact for Queues

1.1 Features of the artefact

  • Artefact tests user's understanding on basic operations of queues.
  • Artefact generates a random question and two Queues (Queue A and Queue B).
  • User should use the artefact to solve the question.
  • User can Submit answer and get feedback(right/wrong)

2.2 Body Section Elements

2.2.1 Instruction Box

This is an instruction box, in which the instructions needed for the user to perform the experiments are displayed in a form of list. 

<div class="instruction-box">
  <button class="collapsible">Instructions</button>
  <div class="content">
    <ul>
      <li><b>New question</b> generates Queue A with a set
        of random numbers and an empty Queue B</li>
      <li>You need to dequeue the elements of the Queue A
        in Ascending order/Descending order based on
        question given, using Queue B.</li>
      <li><b>Dequeue A</b> dequeues the element from queue
        A and <b>Dequeue B</b> dequeues the element from
        queue B</li>
      <li><b>Enqueue to A</b> dequeues the element from
        queue B and enqueues it to queue A and <b>Enqueue to
          B</b> dequeues the element from queue A and enqueues
        it to queue B</li>
      <li><b>Reset</b> restarts the same question
        again</li>
      <li>To check the sequence dequeued is correct or not
        click on <b>Submit</b> button and feedback is
        displayed in comments box
      </li>
    </ul>
  </div>
</div>
