1 Exercise artefact for Stacks

1.1 Features of the artefact

  • Artefact tests user's understanding on basic operations of stacks.
  • Artefact generates a random question and two stacks (stack A and stack B).
  • User should use the artefact to solve the question.
  • User can Submit answer and get feedback(right/wrong)

2 HTML Framework of Stack Exercise Artefact

2.1 Head Section Elements

Title, meta tags and all the links related to Js, CSS of this artefact comes under this section. 

<title>Stacks Exercise</title>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<link rel="stylesheet" href="../css/stacks-and-queues.css">

2.2 Body Section Elements

2.2.1 Instruction Box

This is an instruction box, in which the instructions needed for the user to perform the experiments are displayed in a form of list. 

<div class="instruction-box">
  <button class="collapsible">Instructions</button>
  <div class="content"> 
    <ul>
      <li>You are given two stacks: A and B.</li> 
      <li>Stack A contains a random set of numbers.</li>
      <li>You are also able to do the following:<br>Pop from
      any one stack and push to the other.<br>Pop from any
      stack. <br>If you pop from a stack, the number will
      output out to the screen.</li>
      <li>Output the numbers in ascending order. </li>
    </ul>
  </div>
</div>

2.2.2 Question Holder

This will hold the question of Queue Exercise artefact. 

<div id="question">
  <b>Question:</b> A set of elements are given in Stack
  A. With the help of Stack B, <b>POP</b> the elements in
  ASCENDING order.
</div>
